■ [Ruby] Nested Hash

これはなかなか面白い。 

% irb
>> hash = Hash.new{|h, k| h[k] = Hash.new(&h.default_proc)}
=> {}
>> hash[1][2][3] = 1
=> 1
>> hash[:a][:b][:c] = 1
=> 1
>> hash
=> {:a=>{:b=>{:c=>1}}, 1=>{2=>{3=>1}}}
