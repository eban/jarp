2008-10-12 (Sun)

■ [Linux] soxコマンド

mp3とかoggとか音量を正規化したくてsoxコマンドを使おうと思ったら、 

% sox -h|grep -o mp3
%

mp3はないらしい。oggはある。
mp3ってことはやはり大人の事情でFedoraはdisable状態なのかな。 それはそれとして、 

% sox foo.ogg -e stat -v

で表示される値を与えれば正規化できる。 

% sox -v`sox foo.ogg -e stat -v` foo.ogg foo-norm.ogg

てな具合で。

2008-10-20 (Mon)

■ [Firefox] Fx3でcookies.txt形式にexport

Fxのcookie使ってwgetしようと思ったら、Fx3だとsqliteになってるじゃん。
というわけで探しまくって Export Cookies という拡張を入れてみた。 

% sqlite3 /path/to/cookies.sqlite \
  'SELECT isHttpOnly, host, path, isSecure, expiry, name, value FROM moz_cookies ORDER BY id DESC'

の結果をいじってcookies.txtの形式に変換すればいいんだろうけど、
Fxを立ち上げたままだとSQL error: database is lockedと言われてしまう。
