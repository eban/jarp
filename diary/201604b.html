2016-04-19 (Tue)

■ bash compound-command

man bashすると関数は 

name () compound-command

のようにcompound-commandになっている。
compound-commandはcommandじゃないのでこういう違いがある。 

% sh -c 'foo()echo yes;foo'
yes
% bash -c 'foo()echo yes;foo'
bash: -c: 行 0: 予期しないトークン `echo' 周辺に構文エラーがあります
bash: -c: 行 0: `foo()echo yes;foo'

bashでは{}とかで囲む必要がある。
