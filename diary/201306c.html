2013-06-27 (Thu)

■ じぇじぇ

jsonにjjがあることを知った。今まではppで代用していたが、これからはjjだ。 

% echo '{"id":"abc","foo":"bar","hoge":12345}'|ruby -rpp -rjson -ryaml -e 'pp JSON.parse $<.read'
{"id"=>"abc", "foo"=>"bar", "hoge"=>12345}
% echo '{"id":"abc","foo":"bar","hoge":12345}'|ruby -rjson -ryaml -e 'jj JSON.parse $<.read'
{
  "id": "abc",
  "foo": "bar",
  "hoge": 12345
}
