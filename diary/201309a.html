■ gawkとmawkの違い

gawkとmawkの違いはいろいろあるが、今日気づいた違い。 

% gawk 'BEGIN{n=1; a[1] = 1; a[2] = 2; a[++n] = a[n]; print a[1], a[2]}'
1 1
% mawk 'BEGIN{n=1; a[1] = 1; a[2] = 2; a[++n] = a[n]; print a[1], a[2]}'
1 2

つまりgawkは右辺が先、mawkは左辺が先に評価されるってことだな。

■ 改行のない行をread

n sectionはなぜか最終行に改行がないのでBashでreadすると困る。
readは改行がないとエラーになるので。
しかたないので前処理で最終行に改行を入れるわけだけど、さてどうするか。

最初に思い浮かぶのはsedで$s/$/\n/という感じだが、
もっと簡単に$Gだと余計にもう1個ついてしまう。
で、もっとも短いのはawk 1になるわけだけど、
これは{print}と同じだから。
printは改行ありなので、awk 1は厳密にはcatと同じにはならない。

2013-09-08 (Sun)

■ 意味がなかったdelicious監視

diigoからdeliciousへ同期してるかを監視しなきゃいけないのに、
iftttで日記のRSSからdeliciousへ送ってた。これじゃ全然意味ない。
やっぱprivateなfeedが必要なので、secret keyを調べたというかぐぐった。
How To Find the Private Key For the Delicious API
まさにその通りの記事で、http://feeds.delicious.com/{username}/ にアクセスして、
右側のPrivateというリンクのprivate=以下のkeyを使えばいいらしい。 

% curl -s http://feeds.delicious.com/v2/json/eban\?private\={key} >eban.json               
% jq '.[0]["dt"]' eban.json
"2013-09-08T16:09:13Z"

いけたようだ。

■ Xfce4 ジェネリックモニター

Xfce4のパネルにコマンドの結果を表示できないものかと探していたら見つかった。
インストールはsudo aptitude install xfce4-genmon-pluginでok。
プロパティの"コマンド"に実行したいコマンドを書けばいいが、
シェルを経由しないので、パイプを使うときは 

sh -c 'foo|bar'

のように書く必要がある。
実行する間隔も設定できるのでニュースのトピックスなんかをランダムに表示するといい感じだ。
