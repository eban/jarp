2007-05-13 (Sun)

■ [Vim] Stable Vim version 7.1 has been released

Announcing: Vim (Vi IMproved) version 7.1
This is a stable release of Vim, version 7.1. Since version 7.0 lots of problems were fixed and runtime files were updated. It has been one year and five days since 7.0!
Most of Vim 7.1 was already available as patches for quite a while. A couple of test versions were made to spot problems in the distribution. Therefore Vim 7.1 can be considered very stable.
If you are using an older version, it is highly recommended you install Vim 7.1. Several crashing bugs and a security issue were fixed.
Once you have installed Vim 7.1 you can find details about the changes since Vim 7.0 with ":help version-7.1".

2007-05-16 (Wed)

■ [Linux] Fedora Core 6をネットワークインストール再挑戦

またCD-ROMドライブしかないPCへFedora Core 6をインストールすることになった。
前回の反省でhttpじゃなくてftpならいけるはずだと思い、
ちょうどいい機会なので試す。
うちの会社のFTPゲートウェイは 

ユーザ名: ユーザ名@ftpサイトのホスト名
パスワード: パスワード

という感じで指定するタイプなので、 

FTPサイト名: FTPゲートウェイ
Fedora Coreディレクトリ: /pub/linux/fedora/core/6/i386/os

としてUse non-anonymous userをcheckしておく。あとは 

ユーザ名: anonymous@ftp.iij.ad.jp
パスワード: メールアドレス

てな感じで簡単にいけた。

あとboot promptでip=,netmask=,gateway=を指定しなければ全部聞いてくるので、
無理に指定しないほうがよさげ。

■ [Firefox] FoxTorrent

以前FC6でFoxTorrent 1.0を試したら全然動かなくて残念な思いをしたが、
Open Tech Pressの記事 を見ると1.03ではいけてるようだ。
早速ためしてみるとRedSwoosh deamonを何度も起動しようとして失敗してる様子。
psしてもたしかに起動してない。lddしてみたらlibssl.so.0.9.8にリンクしてる。

libcryptoも同様か。これでなんとか起動できるようになった。
でもなぜかダウンロードはしてくれないのであった。残念。
また半年ほど寝かせよう。
