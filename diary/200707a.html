2007-07-07 (Sat)

■ [IRC] ircIIの日本語設定

ircIIの日本語設定は~/.ircrcに 

set input_encoding EUC-JP
set display_encoding EUC-JP
set irc_encoding ISO-2022-JP

という感じで。
端末がUTF-8な場合はinput_encodingとdisplay_encodingをUTF-8にすればok。
