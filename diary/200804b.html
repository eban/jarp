2008-04-16 (Wed)

■ [Git] GIT_PROXY_COMMAND

この記事 を見て早速 

% git clone git://github.com/rails/rails.git

と試してみたが、git:だとfirewallが邪魔してアウト。調べてみると(git-config(1)) 

#! /bin/sh
corkscrew httpのproxyサーバホスト名 ポート番号 $1 $2

のようなshell scriptを作っておけば 

% GIT_PROXY_COMMAND=$HOME/bin/git-proxy.sh git clone git://github.com/rails/rails.git

でいける。corkscrewはcygwinにもパッケージとして存在するので便利。
