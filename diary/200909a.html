今日はこんなのを思いついた

% sleep 10
% end=`date +%s`
% date --utc --d @$[end-start] +%H:%M:%S

エポック秒で考えると意外に簡単。GNU coreutilsのdate限定かな。

time sh <<EOT
...
EOT

ってのもなんだし。あと自由に時間の形式も指定できない。

■ [Golf] Negatenary

負数で割った余りって言語によって違うから面倒だよねえ。 

% awk 'BEGIN{print 1%-3}' 
1
% ruby -e 'p 1%-3' 
-2
% perl -le 'print 1%-3'  
-2
% awk 'BEGIN{print -1%3}' 
-1
% ruby -e 'p -1%3' 
2
% perl -le 'print -1%3' 
2
