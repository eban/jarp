2010-05-24 (Mon)

■ [Ubuntu] VirtualBoxのインストール

Ubuntu 10.04へVirtualBox 3.2.0をインストールした。
まず/etc/apt/sources.listに 

deb http://download.virtualbox.org/virtualbox/debian lucid non-free

を追加し、次にパッケージのキーを取得する。 

% wget -q http://download.virtualbox.org/virtualbox/debian/oracle_vbox.asc -O- | sudo apt-key add -

sun_vbox.ascからoracle_vbox.ascに変わっているので注意。あとは普通にインストール。 

% sudo aptitude update
% sudo aptitude install virtualbox-3.2

2010-05-28 (Fri)

■ [Linux] lesskeyでlessのline editをemacs key bindに

lessで検索しようとして数文字入力した後に^Bして残念な思いをすることがあるが、
このあたりはlesskeyで設定できる。~/.lesskeyに 

#line-edit
^B left
^F right
^A home
^E end
^D delete
^P up
^N down

と書いてlesskeyを実行すると~/.lessができる。これで準備OK。あとは定義したキーが使えるようになる。
