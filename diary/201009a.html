2010-09-08 (Wed)

■ [DNS] dyndns.comの30日で来る警告

IPアドレスが変わったらdyndns.comの更新をするようなスクリプトにしてたら、
30日経ったらそろそろexpireするよとメールが来た。
そうか、そっちも条件に入れないとだめなんだな。
というわけで、findで判断することに 

% find ip.txt -mtime +30 -print

で、ip.txtが更新されて30日以上経ってればip.txtと表示される。

2010-09-10 (Fri)

■ fibの高速化

再帰しなきゃいいんですよ。 

def fib(n)
  a = b = 1
  n.times{a = b + b = a}
  b
end

これならfib(10000)もok。
