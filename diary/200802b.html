■ [Subversion] version.hだけcheckout(実際はupdate)する

なかださんからの情報で、こんな手順でversion.hだけcheckoutできることがわかった。

% cd version
% mkdir .svn
% ruby -e 'u="svn+ssh://svn@ci.ruby-lang.org/ruby";print "8\n\ndir\n1\n#{u}/trunk\n#{u}\n\f\n"'>.svn/entries
% mkdir -p .svn/tmp;svn cleanup;cp -a .svn/tmp/* .svn
% svn up version.h

2008-02-20 (Wed)

■ [Golf] Duplicate characters

sedの解がみんな違うのも面白い。
awkは最初 

gsub(/./,"&&")&&$0=$0RS

としてたが、 

gsub(/./,"&&"),$0=$0RS

でもいいんだよな。たぶん1,0として扱われる。その後 

gsub(/./,"&&");$0=_;1

でいいんだと気づき、最終的には 

gsub(/./,"&&");NF=0_

となった。NF=0_は_が初期化してない変数なのでNF=0 ""と同じ意味。
つまりNF="0"となるので真として扱われる。
NF=.1ってのも秀逸だな。1未満ならいいので.9でもいい。
