2012-03-26 (Mon)

■ Tab Utilities beta版のtab stacksが便利

Firefox: Install Tab Utilities for Opera-esque tab stacksという記事を読んで知ったんだけど、これは便利だ。
stackしたければ他のタブへdropしてやればいい。
移動したいとき困るかというとちゃんと矢印が出てきてstackと移動が選べるようになっている。
ただ勝手に色がついてしまうのは要らないんだけどな。白文字になって見辛い。
extensions.tabutils.colorStackをfalseでいけた。

■ The Exchange Provider add-on is no longer maintainted.

Thunderbirdを再起動したらこんなメッセージがポップアップした。 

The Exchange Provider add-on is no longer maintainted.

You can still use it but new features will not be implemented and
it will not be kept up to date with Lightning and Thunderbird.

Please install it's successor called 'Exchange 2007/2010 Calendar and Tasks add-on'.

After you close this window the add-on page will be displayed so you can install it.

Thank you for using this add-on and we hope you will enjoy the new add-on.

Simon and Michel.

おいおいと思ったら別のアドオンに引き継がれていた。とりあえずdisableにして新しいやつをインストール。
しかし予定が出てこない。ああ、設定を変換しないといけないんだ。右クリックして「Convert to Exchange
2007/2010 Calendar and Tasks add-on」を選ぶと再起動された。しかしやっぱり予定が出てこない。
また右クリックしてみたら、Exchange (EWS) Propertiesってのが現われた。
見てみるとServer URLがhttps://auto/になってた。これじゃどこにもつながりそうにない。
設定どうすればいいのかと調べようと思ったら、Use Exchange's autodiscovery function
というなんとも便利なチェックボックスを発見した。チェックすると
Perform autodiscoveryというボタンが出てくるのでこれをクリック。
なんかちゃんとサーバーを見つけてくれたようで、あとはユーザー名とパスワードを入力して保存。
でもなかなか出てこない。再起動してみたらやっと予定が出てきた。ゲームかよ。

■ エレうたの放送スケジュール変更

毎月最終日曜日の20:05から50分間ということなので早速crontabに登録。 

04 20 * * sun test $(date +\%m) != $(date -d week +\%m) && (sleep 51m;echo -n q) | mplayer -playlist http://mfile.akamai.com/129931/live/reflector:46032.asx -vo null -ao pcm:waveheader:fast:file=/dev/stdout -really-quiet | ffmpeg -y -i - $HOME/radiru/`date +"eleuta-\%Y-\%m-\%d.mp3"`

余裕を見て20:04から51分間。
