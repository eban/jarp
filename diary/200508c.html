■ Dnsmasq 2.23

Changes:
This release has some minor new features, including better DHCPINFORM support and support for dynamic address allocation with BOOTP. There are also a slew on minor bugfixes, mainly in the config-file parser and DHCP subsystem. There are no showstopper bugs that were fixed, so upgrading from 2.22 is not required if that version is working well.
