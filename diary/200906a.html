■ [Fedora] Fedora 10からFedora 11へyumでupgrade

なんかscreen使えって言われた 以外はいつもと同じ。 

% sudo rpm -Uvh \
  http://download.fedora.redhat.com/pub/fedora/linux/releases/11/Fedora/i386/os/Packages/fedora-release-11-1.noarch.rpm \
  http://download.fedora.redhat.com/pub/fedora/linux/releases/11/Fedora/i386/os/Packages/fedora-release-notes-11.0.0-2.fc11.noarch.rpm
% sudo yum update rpm\*
% sudo yum clean all
% sudo yum upgrade

でも会社のPCは最新のFedora 10になっただけだった。なぜ？
