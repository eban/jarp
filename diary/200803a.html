■ [Vim] changelog.vim

\oすると盛大にエラーになってしまう。
/usr/share/vim/vim71/ftplugin/changelog.vimを見るとすぐに原因がわかった。 

" Regular expression used to find a given date entry.
if !exists('g:changelog_date_entry_search')
  let g:changelog_date_entry_search = '^\s*%d\_s*%u'
endif

" Regular expression used to find the end of a date entry
if !exists('g:changelog_date_end_entry_search')
  let g:changelog_date_entry_search = '^\s*$'
endif

こりゃコピペして修正ミスったな。
自前を使ってるからどうでもいいんだけど、 

  let g:changelog_date_end_entry_search = '^\s*$'

に修正しておこう。

■ [Soft] zsync

zsync is a file transfer program. It allows you to download a file from a remote server, where you have a copy of an older version of the file on your computer already. zsync downloads only the new parts of the file. It uses the same algorithm as rsync.
